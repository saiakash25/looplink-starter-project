Welcome to the Looplink starter project!

You can find this page in looplink.ui.base.views. This is the default view.

Go here to see a view using DjangoHtmxActionMixin (a custom utility we built, located in looplink.django_ext.htmx).

In the templates, you will see js_entry being used. This extension is something we created. It's a quick way to define a JavaScript entry point for our Webpack build. See its usage in the looplink.ui.base templates.

The HTMX example uses its own js_entry module to show how to set that up, but you can use base/common_entry if you don't need additional JavaScript.

Any JavaScript files you want to reference in the Webpack build can be found in an assets folder, like the one in looplink.ui.base. Stylesheets can be found in the styles folder located at the project root.

Looking to create an API? Check out the docs for Django REST Framework, or feel free to use a different library.

Lastly, while we have code-organization best practices at Looplink, we're leaving those specifics open-ended for you. :)
